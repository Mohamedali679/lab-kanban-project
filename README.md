Project Description
This project is a proposed Community Lost & Found web application.
It allows users to post and view lost or found items in a shared space,
supporting cooperation and mutual benefit rather than financial exchange.

Sprint 1 Focus
Sprint 1 focuses on planning, collaboration, and project setup.
No application functionality has been implemented at this stage.
